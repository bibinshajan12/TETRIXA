# TETRIXA

Classic Tetris with modern design and smooth drag controls

## ğŸ® Features

- **Smooth Drag Controls**: Intuitive drag-based movement for tetromino pieces
- **Configurable Grid Size**: Customize grid dimensions (8-12 columns, 16-24 rows)
- **Adjustable Game Speed**: 5 difficulty levels from Very Slow to Very Fast
- **Score-Based Level Progression**: Automatic level increases as you score points
- **Ghost Piece Preview**: Visual guide showing where pieces will land
- **Modern UI Design**: Candy Crush-style vibrant interface with soft gradients
- **Dark Gameplay Theme**: Comfortable dark theme for extended gameplay
- **Crystal-Like Blocks**: Glossy, crystal-like block rendering with smooth animations

## ğŸ¯ How to Play

- **Drag Left/Right**: Move the active tetromino horizontally
- **Tap**: Rotate the active tetromino
- **Drag Down**: Soft drop (accelerated fall)
- **Clear Lines**: Complete horizontal lines to score points and level up
